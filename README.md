# 🤖 بوت اختبار أكواد الخصم - مامولا

## 📋 المتطلبات

### 1. Python
- تحميل من: https://www.python.org/downloads/
- تأكد من تثبيت Python 3.8 أو أعلى

### 2. Selenium
```bash
pip install selenium
```

### 3. ChromeDriver (Windows)
- تحميل من: https://googlechromelabs.github.io/chrome-for-testing/
- ابحث عن: **Stable 149.0.7827.115**
- اختر: **win64** أو **win32**
- ضع الملف في نفس مجلد bot.py

---

## 🚀 خطوات التشغيل

### 1️⃣ تثبيت المكتبات
```bash
pip install selenium
```

### 2️⃣ تحميل الملفات
```bash
git clone https://github.com/aseelmaher2hs-ship-it/bot-mamolaa.git
cd bot-mamolaa
```

### 3️⃣ تحميل ChromeDriver
- اذهب لـ: https://googlechromelabs.github.io/chrome-for-testing/
- حمّل النسخة الصحيحة لنظامك (Windows)
- ضعها في نفس المجلد

### 4️⃣ تشغيل البوت
```bash
python bot.py
```

### 5️⃣ خطوات الاستخدام
1. افتح موقع مامولا
2. أضف منتج للسلة
3. اذهب لصفحة الدفع
4. اضغط في حقل الكود
5. اضغط Enter في Command Prompt لبدء البوت
